# Araç Yönetimi Modülü (AracYonetimi)

Bu proje, kurumsal ERP sistemi içerisindeki "Temel İdari İşler - Araç Modülü"nün modern web teknolojileri (.NET 8 ve PostgreSQL) kullanılarak yeniden tasarlanmış halidir.

## 🚀 Kullanılan Teknolojiler
* **Backend:** C# / .NET 8 (ASP.NET Core Web API)
* **Veritabanı:** PostgreSQL 16
* **ORM:** Entity Framework Core 8
* **Dokümantasyon:** Swagger (Swashbuckle)

## 📋 Projenin Amacı
Kurum bünyesindeki aktif, kiralık ve iptal edilmiş araçların takibini yapmak; sözleşme, kilometre, bakım ve sigorta süreçlerini merkezi bir onay mekanizması üzerinden yönetmektir.
